# Cambright Placement Test

This is a placement test platform built with **HTML**, **CSS**, **JavaScript**, and **Express.js**.

## Features

- Welcome page with name/email input
- Grammar, reading, and writing sections
- Automatic scoring for MCQs
- Manual grading for writing
- Admin panel to manage exams
- Anti-cheating measures (planned)

## How to run it locally

```bash
npm install
node index.js
